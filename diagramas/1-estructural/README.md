# Diagramas UML - Estructurales

En esta sección se encuentran los **diagramas UML de tipo estructural**, los cuales representan la **organización y relación entre los elementos del sistema**.

---

## ¿Qué incluye?

Aquí se desarrollarán diagramas como:

* Diagrama de clases
* Diagrama de componentes
* Diagrama de despliegue

---

## Objetivo

El objetivo de estos diagramas es:

* Comprender la estructura del sistema
* Identificar entidades y sus relaciones
* Definir atributos y comportamientos básicos

## Herramientas utilizadas

* PlantUML
* GitHub Actions (validación automática)

---

## Notas

* Todos los diagramas deben estar en formato `.puml`
* Se validan automáticamente mediante GitHub Actions
* No se almacenan imágenes `.png` en el repositorio
